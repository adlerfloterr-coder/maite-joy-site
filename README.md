Crie a pasta do projeto e entre nela
cd maite_joy_site
Inicialize o repositório
git init
Adicione os arquivos
git add
Faça o commit
git commit -m "Site Maitê Joy"
Conecte ao repositório remoto
git branch -M main
git remote add origin https://github.com/SEU-USUARIO/maite-joy-site.git
Envie para o GitHub
git push -u origin main
